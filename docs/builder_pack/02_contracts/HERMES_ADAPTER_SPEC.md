# Hermes Adapter Spec

Heimdal should be usable with NousResearch/Hermes-style persistent agents without becoming dependent on Hermes.

Integration modes:
1. Heimdal as a Hermes Skill.
2. Heimdal as Backend Daemon.
3. Heimdal as Sub-Agent.

Hermes may own long-term social memory and interaction loop. Heimdal owns quality factory, context packets, verification, eval-gated patches, and hardware-aware local execution.

# Hardware Portability

Heimdal must run across CPU-only, Apple Silicon/Metal, NVIDIA CUDA GPUs from 6-128 GB VRAM, 1-8 GPUs, mixed VRAM setups, Ubuntu native, and WSL2.

Minimum: Python 3.11+, 32 GB RAM recommended, SSD-class storage, Ollama backend for beta. GPU optional but strongly recommended.

Use capability profiling, not fixed assumptions.
